# Free vs Premium Comparison

Defo Ads is available in two versions. The free version is a complete, standalone product. Premium adds cloud infrastructure and advanced features.

## Which Version Should I Choose?

### Choose **Free** if you:
- Want a private, local-first tool with no account required
- Are comfortable managing your own OpenAI API key
- Prefer to export campaigns as CSV and upload via Google Ads Editor
- Don't need real-time sync with Google Ads

### Choose **Premium** if you:
- Want to import/export campaigns directly to Google Ads
- Need real-time performance analytics in one place
- Want managed AI without dealing with API keys
- Work with a team and need collaboration features
- Manage multiple Google Ads accounts
- Want automated scheduled syncs

---

## Upgrading from Free to Premium

If you've been using the free version and want to upgrade:

1. Sign up for an account on the Premium version
2. **Export** your data from the free version (JSON format)
3. **Import** it into Premium — your campaigns will be synced to the cloud
4. Connect your Google Ads account to start syncing

Your data migrates seamlessly. Nothing is lost.
