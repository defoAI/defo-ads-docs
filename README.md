# Defo Ads Documentation

## About Defo Ads

- **Free & Open Source** — Use Defo Ads for free with your own OpenAI API key. Your data stays in your browser. No account required.
- **Premium** — Cloud sync, managed AI, Google Ads integration, team collaboration, and more. [Compare plans](getting-started/free-vs-premium.md).
